Problem Overview

2D matrices and grids are fundamental structures in computer science, mirroring real-world scenarios from images and maps to game boards and spreadsheets. Mastering the procedures of working with these structures fosters a deeper understanding of algorithms, data manipulation, and the art of breaking down complex problems into manageable steps.

As you navigate through these matrix and grid problems, you'll not only enhance your programming skills but also cultivate a problem-solving mindset that's ready to tackle a wide range of computational conundrums. Get ready to wield matrices and grids as tools to unlock creative solutions and expand your programming horizons.

Challenge 1

Matrix 90-degrees Rotation

You are given a square matrix of order 'n' (meaning it has 'n' rows and 'n' columns). Your task is to rotate this matrix 90 degrees clockwise in-place. In-place rotation means that you need to modify the given matrix itself without using any additional memory for a new matrix.

Challenge 2

Island Count Problem

You are given a 2D grid where each cell contains either a 0 (representing water) or a 1 (representing land). Your task is to count the number of islands in this grid. An island is defined as a group of connected land cells, where "connected" means horizontally or vertically adjacent.

Challenge 3

MineSweeper Game Logic

Given a 2D grid with characters ('0' to '9' or 'B'), your task is to perform a transformation based on the following rules at a specific (x, y) coordinate:

  • If the character is 'B', the entire grid remains unchanged.
  • If the character is '1' to '9', it should be replaced with '-'.
  • If the character is '0', it should be replaced with '-' and the transformation should propagate to adjacent cells (top, bottom, left, right, and diagonals).
